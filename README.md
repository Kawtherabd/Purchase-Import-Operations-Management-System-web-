# Purchase & Import Operations Management System

## Project Overview
This project is a web-based management system designed to automate and centralize purchasing processes and import operations tracking within a company.

The application aims to improve visibility, organization, and monitoring of procurement activities, from purchase requests to import follow-up, while reducing manual handling and operational errors.

---

## Project Objectives
- Centralize purchase requests and purchasing workflows
- Track import operations and related processes
- Improve monitoring and traceability of operations
- Reduce manual paperwork and repetitive tasks
- Provide a structured and reliable system for decision support

---

## Business Value
This project addresses real-world enterprise needs by:
- Structuring procurement and import processes
- Enhancing operational efficiency
- Improving data consistency and accessibility
- Supporting better coordination between departments

It reflects a business-oriented approach commonly used in enterprise information systems.
